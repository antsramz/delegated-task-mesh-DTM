# Delegated Task Mesh (DTM)
A machine-only mesh for distributing tasks across multiple agents using deterministic routing and capability matching.

---

## What this template does
DTM enables agents to delegate tasks to the correct peer automatically, based on capabilities, load, and routing rules.

---

## How it works
1. System evaluates all agents in the mesh.  
2. Routing rules determine the best-fit agent.  
3. Task packet is forwarded automatically.  
4. Mesh returns a machine-readable confirmation.
